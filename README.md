# SecurePass-Generator

A beginner-friendly Python password generator that creates weak, medium, and strong passwords with custom length support and automatic clipboard copying.

This project is ideal for Python beginners who want to practice working with random values, strings, functions, and basic security concepts.

---

## Features

* Generate weak, medium, and strong passwords
* Custom password length support
* Automatic copy to clipboard
* Simple menu-driven CLI interface
* Lightweight and fast
* Beginner-friendly code structure

---

## Technologies Used

* Python
* random module
* string module
* pyperclip

---

## Installation

### 1. Clone the repository

```bash
git clone https://github.com/your-username/securepass-generator.git
cd securepass-generator
```

### 2. Install dependencies

```bash
pip install pyperclip
```

---

## How to Run

```bash
python password_generator.py
```
